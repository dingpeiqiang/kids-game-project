## 1️⃣ 图片裁剪功能

### 功能描述

为主题封面图添加了专业的图片裁剪功能，支持精确裁剪和美化处理。

### 核心特性

#### 🖼️ 裁剪区域
- **实时预览**：裁剪区域实时显示
- **自适应容器**：自动调整裁剪区域大小
- **高分辨率支持**：最大支持 1920x1080

#### 🎛️ 裁剪设置
- **宽高比选择**：
  - 自由裁剪（不受比例限制）
  - 16:9（标准宽屏）
  - 4:3（传统比例）
  - 1:1（正方形）

- **旋转功能**：
  - 向左旋转 90°
  - 向右旋转 90°

- **缩放功能**：
  - 0.1x - 3.0x 缩放范围
  - 实时预览缩放效果
  - 滑块精确控制

- **重置功能**：
  - 一键重置所有裁剪设置
  - 恢复到初始状态

#### 📊 预览信息
- **实时预览**：裁剪后立即显示效果
- **尺寸显示**：显示裁剪后的宽度和高度
- **高质量导出**：JPEG 格式，0.8 质量

### 使用方式

1. **上传封面图**
   - 在"基本信息"面板上传封面图
   - 支持拖拽上传和点击上传

2. **打开裁剪器**
   - 上传成功后，点击"裁剪图片"按钮
   - 打开裁剪对话框

3. **调整裁剪区域**
   - 选择合适的宽高比
   - 拖动裁剪框调整位置
   - 使用旋转和缩放功能优化效果

4. **确认裁剪**
   - 点击"确认裁剪"按钮
   - 裁剪后的图片自动替换原图

### 技术实现

#### 使用的库
- `vue-cropperjs`：专业的 Vue 3 图片裁剪组件

#### 核心代码
```vue
<ImageCropper
  v-model="cropperVisible"
  :image-url="tempImageUrl"
  :aspect-ratio="16 / 9"
  @confirm="handleCropConfirm"
/>
```

#### 导出格式
- **格式**：JPEG
- **质量**：0.8（80%）
- **最大尺寸**：1920x1080

## 2️⃣ 草稿管理功能

### 功能描述

提供了完整的草稿管理系统，用户可以保存、恢复、删除草稿。

### 核心特性

#### 📋 草稿列表
- **自动保存**：每次保存时自动添加到草稿列表
- **智能排序**：按更新时间降序排列
- **限制数量**：最多保留 10 个草稿

#### 📊 草稿信息
- **主题名称**：显示草稿的名称
- **封面预览**：显示主题封面图
- **更新时间**：智能显示时间（刚刚、X分钟前、X小时前等）
- **文件大小**：显示草稿大小（KB/MB）
- **主题标签**：显示主题的标签

#### 🎛️ 草稿操作
- **恢复草稿**：
  - 一键恢复任意草稿
  - 恢复前有确认提示
  - 恢复后自动加载草稿数据

- **下载草稿**：
  - 下载为 JSON 文件
  - 包含完整的主题数据
  - 支持离线编辑

- **删除草稿**：
  - 删除不需要的草稿
  - 删除前有确认提示
  - 删除后自动更新列表

### 使用方式

#### 保存草稿
1. **手动保存**
   - 点击顶部工具栏的"保存草稿"按钮
   - 自动保存到草稿列表

2. **自动保存**
   - 每 30 秒自动保存一次
   - 避免数据丢失

#### 管理草稿
1. **打开草稿管理器**
   - 点击顶部工具栏的"草稿管理"按钮
   - 打开草稿列表对话框

2. **恢复草稿**
   - 选择要恢复的草稿
   - 点击"恢复"按钮
   - 确认后加载草稿数据

3. **下载草稿**
   - 点击"下载"按钮
   - 下载为 JSON 文件

4. **删除草稿**
   - 点击"删除"按钮
   - 确认后删除草稿

### 技术实现

#### 数据存储
- **存储位置**：`localStorage`
- **存储键**：`gtrs_drafts`
- **数据格式**：JSON 数组

#### 草稿结构
```typescript
interface Draft {
  id: string              // 草稿ID
  name: string            // 草稿名称
  coverImage: string      // 封面图
  themeData: GTRSTheme    // 完整主题数据
  updatedAt: string       // 更新时间（ISO格式）
  size: number           // 数据大小
  tags: string[]         // 主题标签
}
```

#### 核心方法
```typescript
// 保存草稿
saveDraft(theme: GTRSTheme, name: string)

// 加载草稿列表
loadDrafts()

// 恢复草稿
handleRestore(draft: Draft)

// 下载草稿
handleDownload(draft: Draft)

// 删除草稿
handleDelete(draft: Draft)
```
