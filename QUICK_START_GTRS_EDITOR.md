## 📚 快速入门

### 第1步：填写基本信息

在"📋 基本信息"面板中：

1. **主题ID**：系统自动生成（英文+数字+下划线）
2. **适用游戏**：选择目标游戏（如：贪吃蛇、俄罗斯方块等）
3. **主题名称**：输入主题名称（最多20字）
4. **主题封面**：上传封面图片（建议尺寸：1920x1080）

### 第2步：设计全局样式

在"🎨 全局样式"面板中：

1. **配色方案**：
   - 主色（primaryColor）：主要按钮、链接颜色
   - 辅助色（secondaryColor）：次要按钮、标签颜色
   - 背景色（backgroundColor）：游戏背景颜色
   - 文字色（textColor）：主要文字颜色

2. **字体样式**：
   - 字体大小：12px - 32px
   - 字体粗细：400-700
   - 字体类型：无衬线/衬线

3. **快速应用配色预设**：
   - 点击"🌈 彩虹配色" - 明亮活泼
   - 点击"🌙 暗夜配色" - 深色主题
   - 点击"🌊 海洋配色" - 清新蓝色
   - 点击"🌸 樱花配色" - 温暖粉色

### 第3步：上传图片资源

在"🖼️ 图片资源"面板中：

1. **批量上传**：拖拽多个图片到上传区域
2. **分类管理**：
   - 登录页：logo、背景图
   - 游戏页：场景图、UI元素
   - 结果页：胜利/失败图
   - 场景：各关卡场景图
   - UI：按钮、标签、弹窗等

3. **图片预览**：上传后立即显示缩略图和详细信息
4. **拖拽排序**：上下拖动调整资源顺序
5. **编辑信息**：点击编辑按钮修改图片信息

### 第4步：上传音频资源

在"🔊 音频资源"面板中：

1. **上传音频**：拖拽或点击上传音频文件
2. **在线试听**：
   - 播放/暂停/停止控制
   - 进度条（可拖动跳转）
   - 音量控制（0-100%）
   - 循环播放开关
   - 实时波形可视化

3. **分类管理**：
   - 背景音乐（BGM）
   - 音效（点击、胜利、失败等）

### 第5步：预览效果

在"👁️ 实时预览"面板中：

1. **组件预览**：查看按钮、表单、卡片等UI组件样式
2. **游戏场景预览**：查看主题信息和资源统计
   - 主题信息展示
   - 配色方案展示
   - 资源统计（图片数量、音频数量）
   - **提示**：游戏实时预览需要游戏集成 GTRS 主题后才能实现

3. **对比预览**：对比两个主题的差异
4. **移动端预览**：查看主题在移动设备上的效果
   - 多设备模拟（iPhone 14、iPhone SE、iPad、Android）
   - 状态栏模拟
   - 底部导航栏

### 第6步：发布主题

在"🚀 发布主题"面板中：

1. **检查完整性**：系统自动检查必填项
2. **预览主题信息**：查看主题摘要
3. **生成主题JSON**：预览生成的主题配置
4. **提交发布**：点击发布按钮

## 📝 注意事项

### 主题规范

1. **主题ID**：必须是英文字母、数字、下划线的组合
2. **主题名称**：最多20个字符，支持中文
3. **封面图**：建议尺寸 1920x1080，最大 5MB
4. **图片**：支持 JPG、PNG、WEBP 格式，最大 5MB
5. **音频**：支持 MP3、WAV 格式，最大 10MB

### 自动保存

- 编辑器会自动保存草稿（每30秒）
- 也可以手动点击"保存草稿"按钮

### 撤销/重做

- 支持 Ctrl+Z 撤销
- 支持 Ctrl+Y 重做
- 最多记录 50 步操作

## 🔧 技术栈

- **前端框架**：Vue 3 + TypeScript
- **UI组件库**：Element Plus
- **图片处理**：vue-cropper（裁剪）、compressorjs（压缩）
- **音频处理**：原生 Web Audio API（波形可视化）
- **拖拽排序**：vuedraggable
- **样式预处理**：SCSS

## 💡 使用技巧

1. **使用预设配色**：如果没有设计经验，直接应用预设配色方案

## 🆘 常见问题

### Q1: 上传图片失败？
**A**: 检查图片格式（JPG、PNG、WEBP）和大小（最大5MB）

### Q2: 音频无法播放？
**A**: 检查音频格式（MP3、WAV）和大小（最大10MB），确保浏览器支持该格式

### Q3: 预览效果不对？
**A**: 检查配色方案是否正确设置，刷新浏览器重试

### Q4: 主题无法发布？
**A**: 检查完整性检查是否有错误提示，补充缺失的必填项

### Q5: 游戏实时预览为什么是灰色的？
**A**: 游戏实时预览需要游戏集成 GTRS 主题后才能实现，当前版本只展示主题信息
