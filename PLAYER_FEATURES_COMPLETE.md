## 🔧 技术实现

### 主题切换流程
```typescript
1. 点击主题按钮 → 打开菜单
2. 加载游戏主题列表（API: /api/theme/list）
3. 查找默认主题 → 自动应用
4. 用户选择主题
   ├── 免费主题 → 直接应用
   └── 付费主题 → 检查购买 → 未购买 → 显示购买确认
5. 确认购买 → API: /api/theme/buy
6. 下载主题配置 → API: /api/theme/download
7. 应用主题 → postMessage 给游戏 iframe
8. 保存到 localStorage
```

### 与游戏通信
```typescript
// 发送主题配置到游戏
gameFrame.value.contentWindow.postMessage({
  type: 'THEME_CHANGE',
  theme: theme,
}, '*');

// 游戏内监听
window.addEventListener('message', (event) => {
  if (event.data.type === 'THEME_CHANGE') {
    applyTheme(event.data.theme);
  }
});
```
