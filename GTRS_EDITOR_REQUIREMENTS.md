# GTRS 主题编辑器 - 功能需求文档

## 📋 目标定位

**用户**：普通玩家（非专业开发者）
**核心目标**：零代码、可视化、易上手、功能强大
**核心场景**：玩家可以自己创建、编辑、上传游戏主题

---

## 🎯 核心需求（您的想法）

### ✅ 已确认需求
- 🖼️ **图片预览** - 上传图片后立即预览效果
- 🔊 **声音在线听** - 上传音频后可以播放试听
- 🎨 **功能强大** - 支持完整的主题自定义
- 👶 **易用性强** - 普通玩家也能轻松使用

## 📋 功能模块需求

### 模块1：基本信息（📋）
- [ ] 主题名称（支持中文，最多20字）
- [ ] 主题描述（支持富文本，最多200字）
- [ ] 主题封面图（上传预览，自动裁剪为16:9）
- [ ] 选择适用游戏（贪吃蛇、植物大战僵尸、其他）
- [ ] 主题标签（多选：可爱、复古、科幻、卡通等）

### 模块2：全局样式（🎨）
- [ ] 主色调（颜色选择器 + 预设色板）
- [ ] 辅助色（颜色选择器）
- [ ] 背景色（颜色选择器）
- [ ] 文字颜色（颜色选择器）
- [ ] 字体选择（下拉菜单）
- [ ] 圆角大小（滑块）
- [ ] 阴影效果（开关 + 选择器）
- [ ] 实时样式预览（预览卡片、按钮、表单等）

### 模块3：图片资源（🖼️）
- [ ] **图片预览** - 上传后立即显示预览图
- [ ] 5个分类管理（登录、场景、UI、图标、特效）
- [ ] 拖拽上传
- [ ] 图片裁剪（自动或手动）
- [ ] 图片压缩（自动优化大小）
- [ ] 批量上传
- [ ] 图片命名（自动生成英文Key + 中文别名）
- [ ] 图片删除
- [ ] 图片排序（拖拽排序）

### 模块4：音频资源（🔊）
- [ ] **声音在线听** - 上传后可以播放试听
- [ ] 3个分类管理（背景音乐、音效、语音）
- [ ] 音频播放器（播放、暂停、停止、进度条）
- [ ] 音量调节（滑块）
- [ ] 音频信息显示（时长、大小、格式）
- [ ] 批量上传
- [ ] 音频命名（自动生成英文Key + 中文别名）
- [ ] 音频删除

### 模块5：实时预览（👁️）
- [ ] 主题预览卡片（展示整体效果）
- [ ] 组件预览（按钮、卡片、表单等）
- [ ] 游戏场景预览（模拟游戏中的实际效果）
- [ ] 对比预览（默认主题 vs 当前主题）
- [ ] 移动端预览（手机端效果）

### 模块6：发布主题（📤）
- [ ] 完整性检查（检查必填项）
- [ ] Schema 校验（自动校验JSON格式）
- [ ] 提交审核
- [ ] 发布到市场
- [ ] 设置价格（免费或付费）

### 易用性设计
- [ ] **新手引导** - 第一次使用时显示引导教程
- [ ] **操作提示** - 每个功能都有清晰的提示文字
- [ ] **快捷操作** - 常用功能一键完成
- [ ] **撤销/重做** - 支持操作撤销和重做

- [ ] **上传进度条** - 显示上传进度
- [ ] **加载动画** - 加载中显示动画
- [ ] **成功/失败提示** - 操作结果即时反馈
- [ ] **错误提示** - 友好的错误提示信息
