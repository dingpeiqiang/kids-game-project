## 🎮 完整用户流程

### 玩家：浏览和购买主题

```
1. 访问主题商店页面
   └── /admin/themes/store

2. 浏览主题
   ├── 搜索关键词
   ├── 筛选条件（范围/价格/排序）
   ├── 翻页浏览
   └── 查看主题详情

3. 获取主题
   ├── 免费主题 → 点击"🎁 领取"
   └── 付费主题 → 点击"💰 购买" → 确认 → 完成

4. 使用主题
   └── 点击"✅ 使用" → 应用到游戏
```

### 管理员：管理主题

```
1. 访问管理后台
   └── /admin/themes

2. 创建主题
   ├── 填写基本信息
   ├── 上传主题配置
   ├── 设置价格
   └── 选择适用范围

3. 管理游戏关联
   └── /admin/games → 🎨 → 添加/移除主题

4. 查看统计数据
   └── 下载次数/收益/评分
```
