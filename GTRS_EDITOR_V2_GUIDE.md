## 🚀 如何使用

### 1. 访问编辑器

**URL**：`http://localhost:3000/admin/gtrs-theme-creator-v2`

或者通过管理后台导航进入。

### 2. 创建主题流程

#### 第1步：填写基本信息（📋）
- 选择适用游戏（贪吃蛇、植物大战僵尸等）
- 填写主题名称（支持中文）
- 上传主题封面图（建议尺寸 1920x1080）
- 选择主题标签（可爱、复古、科幻等）
- 填写主题描述

#### 第2步：设计全局样式（🎨）
- 设置主色调、辅助色、背景色、文字颜色
- 选择字体（微软雅黑、思源黑体等）
- 调整字体大小、粗细
- 设置圆角大小和阴影效果
- 应用预设配色方案（可选）

#### 第3步：上传图片资源（🖼️）
1. 选择分类（登录页、场景、UI元素、图标、特效）
2. 点击"上传图片"按钮
3. 拖拽或点击上传图片（支持批量）
4. 图片自动压缩和预览
5. 自动生成英文 Key 和中文别名
6. 可拖拽排序、编辑、删除图片

#### 第4步：上传音频资源（🔊）
1. 选择分类（背景音乐、音效、语音）
2. 点击"上传音频"按钮
3. 拖拽或点击上传音频（支持批量）
4. 自动获取音频时长和文件大小
5. 点击播放按钮试听音频
6. 可调节音量、循环播放
7. 可编辑音频信息（Key、别名、音量）

#### 第5步：预览和发布（📤）
1. 查看实时预览（组件预览、游戏场景预览、对比预览、移动端预览）
2. 检查完整性（自动检测必填项和资源）
3. 选择发布选项（免费或付费）
4. 提交发布

## 📝 注意事项

### 1. 图片要求
- 格式：PNG、JPG、GIF、WEBP
- 大小：≤ 5MB
- 建议尺寸：1920x1080（16:9）
- 自动压缩到 80% 质量

### 2. 音频要求
- 格式：MP3、WAV、OGG
- 大小：≤ 10MB
- 自动获取音频时长
- 支持循环播放

### 3. 主题要求
- 主题名称：2-20字
- 主题描述：≤200字
- 至少包含 1 张图片
- 至少包含 1 个音频

### 4. 浏览器兼容性
- Chrome 90+ ✅
- Firefox 88+ ✅
- Safari 14+ ✅
- Edge 90+ ✅
